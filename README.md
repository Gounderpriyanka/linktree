# 🏎️ Link Tree — Racing Edition

A modern, animated Link Tree website designed to centralize and share my professional online presence. Built using **HTML**, **Tailwind-style CSS**, **Font Awesome**, and **vanilla JavaScript**, this page helps others easily explore my work, certifications, and profile across platforms — wrapped in a racing-inspired visual theme.

---

## ✨ Features

✔ Racing-inspired "pit pass" card design with red & blue livery accents  
✔ Animated **#95 race car** driving across the screen — spoked wheels, headlight glow, exhaust sparks, and speed trails  
✔ Scrolling road strip with moving lane markings for motion realism  
✔ Ambient animated background — soft glow orbs, drifting speed-line texture, and rising ember particles  
✔ Ripple tap/click animation on every link button  
✔ Scroll-triggered reveal animations, staggered for a smooth entrance  
✔ Glassmorphism card with gradient border and rotating avatar ring  
✔ Mobile responsive layout  
✔ Respects `prefers-reduced-motion` for accessibility

## 🚀 Deployment

This project can be hosted easily using **GitHub Pages**.

Steps:
1. Push files to a GitHub repository
2. Go to Settings → Pages
3. Select **main branch → root**
4. Save and wait for deployment link
